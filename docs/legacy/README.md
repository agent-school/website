## What This Folder Contains

Docs in this folder describe the **original Agent School website** before the BMAD complete redesign (2026). The original design used:

- **Color palette:** Navy, Amber, Cream, Charcoal
- **Aesthetic:** Academic, warm, cream backgrounds
- **Typography:** Fraunces (display), DM Sans (body), JetBrains Mono (code)
